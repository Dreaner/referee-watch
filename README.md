# RefereeWatch ⌚️⚽

**RefereeWatch** is an Apple Watch app designed for football referees, helping manage match time, record key events, and keep track of scores effortlessly.

## Features
- **Precise Timer**: Displays match time in minutes:seconds.hundredths (mm:ss.xx)
- **Live Score**: Home and away scores update in real-time
- **Event Recording**:
  - Goals (Normal, Penalty, Own Goal)
  - Yellow / Red Cards
  - Substitutions (player off / player on)
- **Input Options**: Digital crown picker and voice input for quick player number selection
- **Clean UI**: Event buttons show only icons for minimalistic interaction

## Technical Details
- Built with **SwiftUI + Combine**
- Supports **Apple Watch** previews
- Modular architecture: Models, Managers, and Views separated for easy maintenance and future expansion
